# Tasks: Typography system

## Pasos

- [ ] Abrir `config/settings_data.json`
- [ ] Localizar la propiedad `type_header_font`
- [ ] Asegurar que su valor sea `assistant_n4`
- [ ] Localizar la propiedad `type_body_font`
- [ ] Asegurar que su valor sea `assistant_n4`
- [ ] Localizar `heading_scale` y poner valor `110`
- [ ] Localizar `body_scale` y poner valor `100`
- [ ] Guardar el archivo
- [ ] Verificar en `http://127.0.0.1:9292` que no hay errores de `font_url` en la terminal
- [ ] Verificar que los títulos se ven ligeramente más grandes que el body

## Validación

No deben aparecer errores tipo `font_url can only be used with a font drop` ni `'X' is not a valid font handle` en la terminal.
